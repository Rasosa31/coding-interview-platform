# Crear Repositorio en GitHub

## Pasos para crear el repositorio:

1. Ve a [github.com/new](https://github.com/new)
2. Nombre del repositorio: `coding-interview-platform`
3. Descripción (opcional): "Platform for online coding interviews with real-time collaboration"
4. **NO marques** "Initialize with README" (ya tenemos uno)
5. **NO marques** "Add .gitignore" (ya tenemos uno)
6. **NO marques** "Choose a license" (opcional)
7. Haz clic en **"Create repository"**

## Después de crear el repositorio:

Una vez creado, ejecuta este comando en la terminal:

```bash
cd /Users/ramirososa/coding-interview-platform
git push -u origin main
```

O si prefieres, puedo hacerlo por ti después de que crees el repositorio.
